Label Elemen

HTML label element digunkaan untuk merepresentasikan sebuah judul atau penamaan untuk item control yang biasanya berkaitan dengan sebuah form control.

Codingan
<!DOCTYPE html> <html> <head> <title>Label Elemen</title> </head> <body> <h1 title="Dewan Perwakilan Rakyat">DPR</h1> </body> </html>
Hasil codingan

Analisis
Gambar diatas merupakan script untuk membuat label elemen yang nantinya akan memiliki efek munculnya tool tip pada browser ketika kursor didekatkan pada elemen tersebut. Terlihat bahwa yang ada dalam gambar diatas adalah elemen DPR.
